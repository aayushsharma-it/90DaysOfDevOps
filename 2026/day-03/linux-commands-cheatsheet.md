# Linux Commands Cheat Sheet

## File System Commands

| Command | Usage |
|----------|-------|
| mkdir | Create new directory |
| ls | List files and folders |
| pwd | Show current directory |
| touch | Create new file |
| cd | Change directory |
| clear | Clear terminal screen |
| rm | Delete file |
| rm -r | Delete folder recursively |
| cat | View file content |
| echo | Print/write text |
| head | Show first lines of file |
| tail | Show last lines of file |
| less | View large files page by page |
| cp | Copy files |
| cp -r | Copy folders recursively |
| mv | Move or rename files |
| wc | Count words, lines and characters |
| ln | Create links between files |
| cut | Extract specific columns/text |
| tee | Save output to file and display |
| sort | Sort file content |
| diff | Compare two files |

---

## Disk Usage Commands

| Command | Usage |
|----------|-------|
| df -h | Show disk space usage |
| free -h | Show memory usage |
| uptime | Show system uptime |

---

## Process Management Commands

| Command | Usage |
|----------|-------|
| ps | Show running processes |
| top | Live process monitoring |
| kill | Stop process using PID |

---

## Networking Commands

| Command | Usage |
|----------|-------|
| netstat | Show network connections |
| ping | Check internet connectivity |
| ifconfig | Show network configuration |
