# My 90 Days of DevOps Learning Plan

## Current Level
I am a student starting my DevOps journey and learning cloud and automation technologies step by step.

## My Understanding of DevOps
DevOps is the practice of improving collaboration between development and operations teams using automation, CI/CD, cloud, monitoring, and infrastructure tools.

## Why I Am Starting DevOps
I want to build practical skills, improve my problem-solving abilities, and become industry-ready for software and cloud engineering roles.

## My 3 Goals for the Next 90 Days
1. Learn Linux, Git, and Docker fundamentals
2. Build CI/CD pipelines using GitHub Actions or Jenkins
3. Deploy applications using Kubernetes and cloud platforms

## 3 Core DevOps Skills I Want to Build
- Linux troubleshooting
- CI/CD automation
- Containerization and Kubernetes

## Weekly Time Commitment
- Weekdays: 4-5 hours daily
- Weekends: 2–3 hours daily

## How I Will Stay Consistent
I will study every day, document my learning publicly on LinkedIn and GitHub, and focus on consistency instead of perfection.
